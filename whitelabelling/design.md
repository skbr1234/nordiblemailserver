# Nordible Email Website Design System
Version: 1.0  
Product: Nordible Professional Email  
Website: email.nordible.co

---

# 1. Purpose

The website should position Nordible Email as:

- Professional
- Trustworthy
- Secure
- Premium yet affordable
- Friendly and approachable

The mascot should create brand recall and emotional connection without making the product feel childish.

Design goal:

> “Professional SaaS with a memorable mascot.”

Reference feeling:

- Linear → premium spacing
- Slack → friendly
- Notion → clarity
- Duolingo → mascot personality
- Stripe → conversion-focused SaaS UX

---

# 2. Core Website Goals

Primary Goals:

1. Convert visitors into paid email customers
2. Build trust for business email hosting
3. Reduce fear around migration/setup
4. Explain technical concepts simply
5. Differentiate from generic hosting providers

Primary CTA:

**Get Professional Email**

Secondary CTA:

**View Pricing**

# 3. Brand Personality

Tone:

- Friendly
- Confident
- Helpful
- Trustworthy
- Clear

❌ Corporate jargon

# 8. Mascot Usage System

Mascot Purpose:

The mascot acts as:

> Brand guide + trust builder

NOT decoration.

---

## Mascot Rules

DO:

✅ Hero section usage  
✅ Explain benefits  
✅ Security reassurance  
✅ Pricing recommendation  
✅ Empty states  
✅ FAQ helper

DON'T:

❌ Put mascot everywhere  
❌ Repeat same pose  
❌ Make UI childish  
❌ Distract from CTA

---

## Required Mascot Variations

1. Hero pose (thumbs up)
2. Security pose
3. Laptop working pose
4. Mail sending pose
5. Pricing recommendation pose
6. Celebration pose
7. Support/helper pose
